# vue-link-shortener
Optimize the length of a website url 

## Project setup
```
yarn
```

### Compiles and hot-reloads for development
```
yarn
```

### Compiles and minifies for production
```
yarn build
```

### Customize configuration
See [Configuration Reference](https://vitejs.dev/config/).

### Dependencies
- bootstrap
- vue
- vue-tsc
- vite
- @vitejs/plugin-vue
- @vueuse/core
